# Source properties

## inparam.time_src_recv

__time__:

time section set to default,

time_record_length - 100.0

__source__:

src type - gauss

could be erf, delta, gauss

source_stf_half_duration - 10.0

__stations__:

stations file - STATIONS

everything else set to default
